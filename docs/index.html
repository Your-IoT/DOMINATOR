About RFID DOMINATOR

The RFID DOMINATOR is a cutting-edge domination stopwatch designed for Airsoft, Paintball, and other competitive sports, adding a new level of strategy and excitement to your gameplay. This innovative device has onboard display, that tracks the total occupation time of a control point, making it an essential tool for organized events and team-based competitions. Using RFID technology, players can occupy the point by simply tapping their RFID wristband or card to the device. The stopwatch will beep, if there is changed active team on the point and time is counting for this team, until a player from the opposing team claims the point, or if referee pauses the time. A referee can also reset the time, if needed, ensuring smooth and fair play gameplay experience. The device supports standard gameplay with 2 teams and offers the flexibility to include up to 4 teams in a single match.

Usage Scenarios

Ideal for playgrounds, fields offering Airsoft / Paintball experience. Events such as LARP, MilSim, Scenario, Speedball, due the flash size that supports hundrets of RFID tags in a single application

    PCB

    RFID DOMINATOR 2.0 PCB specifications

    104 x 103 x 1,6 mm, double-sided FR4 type PCB. Silkscreen on the PCB marks position and orientation of components, how these need to be soldered.

    Open-source hardware support

    Hardware compatible with open-source hardware

    RFID DOMINATOR's hardware is related to open-source hardware usage. It is using Arduino modules and Arduino board itself for modular design and its advantages of open-source hardware (cost-effective, easy to replace).

    Assembled PCB

    View on the PCB, after it was assembled by compatible components

    After the RFID DOMINATOR 2.0 PCB is assembled, total height of the PCB is extended to 45 mm. On the front side, there is LCD2004A screen and MFRC522 reader, alongside with LED diodes and buzzer. From the back side, there is Arduino Nano V3.0 controller.

    Enclosure

    3D printed enclosure for RFID DOMINATOR 2.0

    Enclosure compatible with PCB of RFID DOMINATOR 2.0. Dimensions of the enclosure is 124 x 123 x 47 mm. PCB itself is mounted from the inside, on the front side of the enclosure using 4 pcs of M3 screws. There is possible to use 25 mm (width) tie-belt on the enclosure to mount it on the tree, joist.

    Wiring diagram

    Wiring diagram for RFID DOMINATOR 2.0

    Wiring diagram shows, how Arduino Nano is wired with all other components. For LCD2004A, there is used hardware-wired I2C bus and for MFRC522 SPI-bus. Additional GPIOs are used for LED diodes indications and for buzzer with PWM support.

    Frequently Asked Questions

    The most asked question from the community answered below

    What is energy consumption of the system?

    Standardly near 40 mAh - 60 mAh @ 5V. In terms of power consumption, similar to 0.2 - 0.3 W. You will be able to run this system with 2000 mAh source for 30 - 35 hours (depends on load on the system).

    Can I power it up via powerbank?

    Yes, RFID DOMINATOR can be powered up from powerbank directly via its 5V output.

    How can I upload the firmware?

    Same principle for DEMO or FINAL firmware. You will have .hex file compatible for your Arduino. Upload you can perform using XLoader tool. If you have Arduino Nano with Old Bootloader, your upload speed needs to be set to 57600 baud/s, with New Bootloader it is 11500 baud/s. If software was uploaded successfully, you will receive print about that from XLoader.

    How to copy existing firmware?

    Same file you have received via e-mail and upload to the first unit, you will upload it to the next unit of RFID DOMINATOR. It will work the same way and will react on the same RFID wristbands, cards. You are not limited in count of clones performed.

    Can be team name changed?

    Not at the moment, but soon it will be possible, when AT commands extension will be finished. Team names need to have maximum 3 letters (it will be limited by the software).
