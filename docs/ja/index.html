RFID DOMINATOR について – 競争的なゲームプレイを革新する

RFID DOMINATOR は、エアソフト、ペイントボール、その他の競技スポーツ向けに作られた最先端のドミネーション ストップウォッチで、あらゆる試合の戦略と関与を高めます。

この高度なデバイスには、コントロールポイントの合計占有時間を追跡するオンボードディスプレイが搭載されており、組織的なイベントやチームベースの競技には欠かせないものとなっています。

RFID テクノロジー により、プレーヤーはデバイス上で RFID リストバンドまたはカードをタップするだけで、簡単にコントロールを獲得できます。ストップウォッチがビープ音を鳴らしてチームの変更を知らせ、対戦相手のプレーヤーがポイントを獲得するか、審判がゲームを一時停止するまで、タイマーはアクティブ チームのカウントを続けます。

審判員は必要に応じてタイマーをリセットすることもできるため、シームレスで公正なゲームプレイ体験が保証されます。

多用途性を考慮して設計された RFID DOMINATOR は、2 チーム の標準試合をサポートし、1 回の競技で最大 4 チーム まで拡張できます。

RFID DOMINATOR で競争力を高め、プレイ方法を再定義しましょう。

使用シナリオ

遊び場、エアソフト/ペイントボール体験を提供するフィールドに最適です。LARP、MilSim、シナリオ、スピードボールなどのイベントでは、1つのアプリケーションで数百のRFIDタグをサポートするフラッシュサイズが役立ちます。

サポートされているハードウェア

RFID DOMINATOR 2.0 PCBでは、以下のコンポーネントと互換性があります。

  • Arduino Nano V3.0
  • MFRC522 RFID-RC522 リーダー
  • I2C コンバータを備えた LCD 2004A
  • 7.62 mm ピッチのブザー
  • 5mm LED ダイオード THT
  • 抵抗 THT

    • デモファームウェア

    モジュラー設計、ポータブルファームウェア

    RFID DOMINATOR のモジュール設計により、ユーザーははんだ付けをすることなく主要コンポーネントを交換できます。ホットスワップ方式により、ユーザーは数秒以内にモジュールを取り外して新しいモジュールを装着できます。 ユーザーはファームウェアを任意の数の他のデバイスにコピーして複製できます。一度支払えば、デバイスに必要な回数だけ使用できます。

    プリント基板

    RFID DOMINATOR 2.0 PCB仕様

    104 x 103 x 1.6 mm、両面 FR4 タイプ PCB。PCB 上のシルクスクリーンには、コンポーネントの位置と方向、およびこれらをはんだ付けする方法がマークされています。

    オープンソースハードウェアのサポート

    オープンソースハードウェアと互換性のあるハードウェア

    RFID DOMINATOR のハードウェアは、オープンソース ハードウェアの使用に関連しています。モジュール設計に Arduino モジュールと Arduino ボード自体を使用し、オープンソース ハードウェアの利点 (コスト効率が高く、交換が簡単) を活用しています。

    組み立てられたPCB

    互換性のあるコンポーネントで組み立てられた後の PCB の表示

    RFID DOMINATOR 2.0 PCB を組み立てると、PCB の合計高さは 45 mm に拡張されます。前面には、LED ダイオードとブザーの他に、LCD2004A スクリーンと MFRC522 リーダーがあります。背面には、Arduino Nano V3.0 コントローラーがあります。

    囲い

    RFID DOMINATOR 2.0 用 3D プリント エンクロージャ

    RFID DOMINATOR 2.0 の PCB と互換性のあるエンクロージャ。エンクロージャの寸法は 124 x 123 x 47 mm です。 PCB 自体は、4 本の M3 ネジを使用してエンクロージャの前面の内側から取り付けられます。 エンクロージャに 25 mm (幅) のタイベルトを使用して、木や梁に取り付けることもできます。

    配線図

    RFID DOMINATOR 2.0 の配線図

    配線図は、Arduino Nano が他のすべてのコンポーネントとどのように配線されているかを示しています。LCD2004A の場合、ハードウェア配線の I2C バスが使用され、MFRC522 の場合は SPI バスが使用されます。追加の GPIO は、LED ダイオード表示と PWM サポート付きブザーに使用されます。

    よくある質問質問

    以下は、コミュニティから最もよく寄せられる質問です。

    システムのエネルギー消費量はどれくらいですか?

    システムは通常、5V で 40～60 mAh、つまり 0.2～0.3 W を消費します。2000 mAh の電源では、負荷に応じてシステムを 30～35 時間稼働させることができます。

    パワーバンク経由で電源供給できますか?

    はい、RFID DOMINATOR は、5V 出力を介してパワーバンクを使用して直接電源を供給できます。

    ファームウェアをアップロードするにはどうすればいいですか?

    デモ版ファームウェアでも最終版ファームウェアでも、Arduino と互換性のある提供された .hex ファイルを使用してください。 XLoader ツールを使用してアップロードします。古いブートローダを搭載した Arduino Nano の場合は、速度を 57600 ボー/秒に設定します。新しいブートローダの場合は、115200 ボー/秒を使用します。 アップロードが成功したかどうかは、XLoder からのメッセージで確認できます。

    既存のファームウェアをコピーするにはどうすればいいですか?

    電子メールで受け取った同じファームウェア ファイルを、次の RFID DOMINATOR ユニットにアップロードするだけです。 同じように機能し、同じ RFID リストバンドとカードを認識します。 作成できるクローンの数に制限はありません。

    チーム名は変更できますか？

    現時点ではできませんが、AT コマンドの拡張が完了するとすぐに可能になります。チーム名は最大 3 文字にする必要があります (ソフトウェアによって制限されます)。
