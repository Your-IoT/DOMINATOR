关于 RFID DOMINATOR – Airsoft 秒表

RFID DOMINATOR 是一款先进的统治秒表，专为 Airsoft、Paintball 和其他竞技运动而设计，可提升每场比赛的策略和参与度。

这款先进的设备具有 板载显示屏，可跟踪控制点的总占用时间，使其成为有组织的活动和团队比赛不可或缺的设备。

借助 RFID 技术，玩家只需将 RFID 腕带或卡轻触设备即可轻松获得控制权。秒表会发出哔哔声以指示队伍更换，计时器会继续为活跃队伍计时，直到对方玩家获得分数或裁判暂停比赛。

裁判还可以在必要时重置计时器，确保无缝且公平的游戏体验。

RFID DOMINATOR 专为多功能性而设计，支持两支球队的标准比赛，并可扩展至在单场比赛中容纳最多四支球队。

使用 RFID DOMINATOR 增强您的竞争优势并重新定义您的游戏方式！

使用场景

非常适合游乐场、提供 Airsoft / Paintball 体验的场地。诸如 LARP、MilSim、Scenario、Speedball 等活动，由于闪存大小可在单个应用程序中支持数百个 RFID 标签

支持的硬件

与 RFID DOMINATOR 2.0 PCB 兼容的组件如下

  • Arduino Nano V3.0
  • MFRC522 RFID-RC522 读写器
  • 带 I2C 转换器的 LCD 2004A
  • 蜂鸣器间距为 7.62 毫米
  • 5mm LED 二极管 THT
  • 电阻THT

    • DEMO 固件

    RFID DOMINATOR 提供免费使用的 DEMO 固件，允许您测试设备及其基本功能（如果接线正确，如 RFID DOMINATOR 网页上提供的示意图所示）

    模块化设计，便携式固件

    RFID DOMINATOR 的模块化设计允许用户更换主要组件，而无需焊接任何东西。通过热插拔方法，他可以在几秒钟内移除模块并安装新模块。 用户可以将固件复制和克隆到任意数量的设备上。 只需支付一次许可证，即可根据需要多次使用您的设备。

    PCB

    RFID DOMINATOR 2.0 PCB 规格

    104 x 103 x 1,6 mm，双面 FR4 型 PCB。PCB 上的丝网印刷标记了元件的位置和方向，以及需要如何焊接这些元件。

    开源硬件支持

    与开源硬件兼容的硬件

    RFID DOMINATOR的硬件与开源硬件的使用有关。它使用Arduino模块和Arduino板本身进行模块化设计，并利用开源硬件的优势（成本效益高，易于更换）。

    组装好的 PCB

    使用兼容元件组装后的 PCB 视图

    RFID DOMINATOR 2.0 PCB 组装后，PCB 总高度扩展至 45 毫米。正面有 LCD2004A 屏幕和 MFRC522 读取器，以及 LED 二极管和蜂鸣器。 背面有 Arduino Nano V3.0 控制器。主要组件使用针头。

    附件

    RFID DOMINATOR 2.0 的 3D 打印外壳

    外壳与 RFID DOMINATOR 2.0 的 PCB 兼容。外壳尺寸为 124 x 123 x 47 毫米。 PCB 本身从内部安装在外壳正面，使用 4 个 M3 螺钉。 可以在外壳上使用 25 毫米（宽度）的绑带将其安装在树、托梁上。

    接线图

    RFID DOMINATOR 2.0 接线图

    接线图显示 Arduino Nano 如何与所有其他组件连接。对于 LCD2004A，使用硬件连接的 I2C 总线，对于 MFRC522，使用 SPI 总线。额外的 GPIO 用于 LED 二极管指示和支持 PWM 的蜂鸣器。

    常见问题问题

    以下是我们社区最常见的问题。

    该系统的能耗是多少？

    该系统通常消耗 40–60 mAh @ 5V，相当于 0.2–0.3 W。2000 mAh 电源可运行系统 30–35 小时，具体取决于负载。

    我可以通过移动电源为其供电吗？

    是的，RFID DOMINATOR 可以通过其 5V 输出直接使用移动​​电源为其供电。

    如何上传固件？

    无论是 DEMO 还是 FINAL 固件，请使用提供的与 Arduino 兼容的 .hex 文件。 使用 XLoader 工具上传。对于使用旧 Bootloader 的 Arduino Nano，将速度设置为 57600 波特/秒；对于新 Bootloader，请使用 115200 波特/秒。 XLoader 会发送消息确认上传成功。

    如何复制现有固件？

    只需将您通过电子邮件收到的相同固件文件上传到您的下一个 RFID DOMINATOR 单元即可。 它将具有相同的功能并识别相同的 RFID 腕带和卡。 您可以创建的克隆数量没有限制。

    可以更改团队名称吗？

    目前还不行，但 AT 命令扩展完成后很快就会可以。团队名称最多需要 3 个字母（受软件限制）。
